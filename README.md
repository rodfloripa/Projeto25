# Aprenda sobre doenças com RodMed

Neste projeto foi feita a raspagem de dados do site sobre doenças do Ministério da Saúde e usando
LLM criei uma RAG que reponde perguntas sobre saúde.

O projeto está hospedado no Heroku e se conecta com o Zilliz Cloud, um banco de dados Milvus auto gerenciado.

Utilizei como Sentence Transformer o modelo all-MiniLM-L6-v2, que reduziu bastante o tamanho do container,já
que o limite do Heroku é 2GB.
